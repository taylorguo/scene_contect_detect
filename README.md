# Scene Contect Detect
Using pySceneDetect to segment video into scenes for next application


## 环境 与 依赖项： 
- Python3.6
### basic dependencies 
numpy
opencv-python
imutils
scenedetect

### optional dependencies, used for download video file
requests
urllib3

## 安装
- Python3.6 与 pip 可以百度后自行安装
- 依赖项安装：
pip install -r requirements.txt
